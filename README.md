Python Paint App

A simple interactive drawing application built with Python. This project allows users to draw freely on a canvas using mouse input through a graphical user interface.

---
Features

* Freehand drawing with mouse
* Basic canvas interaction
* Event handling for real-time input
* Simple and clean GUI design

---

Technologies Used

* Python
* Tkinter (GUI library)

---

Installation & Usage

1. Clone the repository:

```bash
git clone https://github.com/your-username/python-paint-app.git
```

2. Navigate to the project folder:

```bash
cd python-paint-app
```

3. Run the application:

```bash
python paint.py
```
